Provides options to disable the commenter's website and deactivate links in comments. Extra protection from spammers, best used with Akismet. 

== Description ==

Antispam Extra provides these additional spam prevention options to Wordpress installations.

1. Prevent commenters from submitting their websites.
2. Disable the links to commenters' websites.
3. Deactivate the links in comments.
4. Treat comments without proper HTTP referer as spam.

Works best for comment spams that get past Akismet.

== Installation ==

1. Upload the 'antispam-extra' folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Go to the Settings page and customize the plugin according to your requirement.
